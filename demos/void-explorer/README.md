# KnowledgeExplorer

> Generated by Void Manifest Engine — A minimal, text-focused knowledge explorer app for browsing ancient Sutras

## Getting Started

```bash
npm install
npx prisma db push
npm run dev
```

## Architecture

- **Stack:** nextjs
- **Entities:** Sutra, SutraRelation
- **Routes:** 3 endpoints
- **Pages:** 2 pages
- **Components:** 6 components
- **Auth:** none

### Data Model
- **Sutra** — Represents an ancient sutra with text and metadata
  - id: integer (required)
  - title: string (required)
  - sanskritText: text (required)
  - englishMeaning: text (required)
  - category: string (required)
  - createdAt: datetime (required)
  - updatedAt: datetime (required)
- **SutraRelation** — Junction table for many-to-many relation between sutras
  - id: integer (required)
  - sutraId: integer (required)
  - relatedSutraId: integer (required)

### API Routes
- `GET /api/sutras` — List all sutras, with optional search and category filter
- `GET /api/sutras/:id` — Get a single sutra by ID, including related sutras
- `GET /api/categories` — List all available categories
